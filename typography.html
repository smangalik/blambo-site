What typography is relevant to this site?

As mentioned previously, the emphasis on fundamental elements means that the smallest possible number of interactive elements are typeset. This means that virtually all the font not in the logo will be generic text in paragraph form as form labels. I find that a font like roboto is best for this setting as it is familiar to most modern environments. It is a sans serif font making it very readible in text on the web. To fulfill the need for both light and heavy font I have selected Roboto Normal 400 and Roboto Bold 700. An additional advantage to using Roboto is its generally vertical aspect ratio which meshes well with the generally vertical columns found on the page.

What typography is used on the logo?

The font used is Opus Plainchord Std with a weight of 400. Opus is a very classical and slightly ornamental serif font which is strong and clean in look making it ideal for a logo.
